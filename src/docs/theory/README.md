# Теоретические материалы

## Бинарные операции

**Бинарная операция** — это операция, принимающая два аргумента и возвращающая один результат.

### Формальное определение
Для множеств A и B, бинарная операция *: A × B → C

### Свойства бинарных операций
- **Коммутативность**: a * b = b * a
- **Ассоциативность**: (a * b) * c = a * (b * c)
- **Существование нейтрального элемента**

## Автоморфные отображения

**Автоморфизм** — это изоморфизм математического объекта на себя.

### Примеры
- Тождественное отображение
- Повороты геометрических фигур
- Симметрии

## Связь с теорией групп

Бинарные операции — основа алгебраических структур:
- Группы
- Кольца
- Поля
