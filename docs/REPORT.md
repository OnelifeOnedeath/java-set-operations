# Отчёт по выполнению задания

##  Условие задания
**Задача:** Задать два целочисленных множества и построить функцию над бинарной операцией.

##  Реализация

### Используемые множества
- **Множество A** = {1, 2, 3, 4, 5}
- **Множество B** = {3, 4, 5, 6, 7}

### Реализованные бинарные операции
1. `(a, b) → a * b + 1`
2. `(a, b) → a² + b²`
3. `(a, b) → |a - b|`

### Ключевые особенности реализации
- **Универсальная функция** `applyBinaryOperation()` - работает с любыми типами данных
- **Functional Interface** - поддержка любых операций через `BinaryOperator<T>`
- **Проверка математических свойств** - замкнутость, коммутативность, нейтральный элемент
- **Табличное представление** - таблица Кэли для операций
- **Комплексное тестирование** - 3 различных теста для проверки корректности
