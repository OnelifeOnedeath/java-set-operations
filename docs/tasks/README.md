# Описания заданий

## Текущее задание

**Цель:** Исследовать бинарные операции на множествах

### Задача
1. Задать два целочисленных множества A и B
2. Реализовать функцию применения бинарной операции
3. Протестировать на различных операциях

### Примеры операций
- Арифметические: +, -, *, /
- Логические: AND, OR, XOR
- Специальные: min, max, НОД

## Дополнительные задания

1. **Проверка свойств** - исследовать коммутативность и ассоциативность
2. **Визуализация** - графическое представление операций
3. **Оптимизация** - улучшение производительности для больших множеств
